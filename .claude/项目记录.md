# 项目记录

## 一、项目是干什么的

这是一个桥接工具，让你能用微信远程指挥电脑上的 Claude Code 干活。

你不在电脑前时，可以在微信里发消息，Claude Code 收到后会在电脑上继续写代码、修 bug、跑测试。写好以后，结果会自动回传到微信。需要你批准某个操作时（比如 Claude Code 要执行一个高风险命令），审批请求也会推送到微信，你回个"yes"就行了。

说白了就是：**微信变成 Claude Code 的遥控器**。

## 四、功能流程说明

### 首次使用（按顺序操作）
1. **装依赖** → 双击 `一键安装依赖.cmd`
2. **扫码登录** → 双击 `一键扫码登录.cmd`，用微信扫终端里的二维码
3. **装 MCP 配置** → 在你用 Claude Code 的项目目录，双击 `一键安装MCP配置.cmd`
4. **检查状态** → 双击 `一键状态检查.cmd`，确认各项都打勾

### 日常使用
1. **启动** → 双击 `一键启动守护进程.cmd`，保持窗口开着
2. **在微信里发消息** → 找到 ClawBot，像正常聊天一样发消息
3. **等待回复** → Claude Code 在电脑上干活，干完回复到微信
4. **审批操作** → 看到审批请求时，回复 `yes <id>` 或 `no <id>`

### 换设备需要重做的事
- 装 Bun（每台电脑一次）
- git clone 拉代码
- 按上面的"首次使用"流程从头走一遍

### 启动流程图
```
安装依赖 → 扫码登录 → 安装配置 → 启动守护进程 → 微信发消息 → Claude 干活 → 微信回复
```
